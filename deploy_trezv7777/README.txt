Деплой на trezv7777.ru (reg.ru)
================================

Содержимое архива залить в корень сайта (public_html или www).

- index.html — админ-панель (открывается по адресу https://trezv7777.ru/ )

Важно: админка обращается к API по адресу /api на том же домене.
Если бэкенд на другом сервере — нужно либо проксировать /api на ваш API,
либо править в index.html строку API_BASE (поиск: "const API_BASE").
